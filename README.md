This project contains a Power BI dashboard created to analyze data and extract insights.

-Power BI desktop
-Excel dataset
## Key insights
-Sum of Boxes Shipped by Product
-Sum of Boxes Shipped by Country
-Sum of Amount by Year, Quarter, Month and Day
-Q and A visual
